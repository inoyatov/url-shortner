# Prerequisits
- python 3.10.4
- venv

# Install
- Create virtual environment
```bash
python3 -m venv --promtp="venv" .venv
```
- Install required modules
```bash
python3 -m pip install -r requirements/development.txt
```

# Run
```bash
cd src/
python manage.py runserver 
```
